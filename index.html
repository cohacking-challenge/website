What is the CoHacking Challenge?

5 weeks
Every Wednesday nights and Sundays
10 JavaScript developers
2 teams
1 project per team
Same Stack : React + Firebase
1 final presentation at the end

How are we going to work?

We will mainly do Mob-Programming
We will document what we will learn and make the code Open Source

Why doing this?

If you don't a technology, you will learn it
If you know it, you will improve with others and by explaining them
It is going to be fun!

What will I learn?

React
Firebase
Building an Open Source project
Good practises from other developers in the room
